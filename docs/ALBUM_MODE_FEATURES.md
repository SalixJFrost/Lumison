# 专辑模式功能详解

## 视觉设计

### 封面尺寸对比
- **桌面端**：480x480px（超大尺寸）
- **平板端**：360x360px
- **手机端**：280x280px

### UI元素层次
```
┌─────────────────────────────────┐
│                                 │
│      [超大专辑封面 480x480]      │
│                                 │
└─────────────────────────────────┘
           ↓
    [歌曲标题 - 大字体]
    [艺术家 - 中字体]
           ↓
    [❤️ 喜欢]  [⋮ 更多]
           ↓
  [时间] ━━━━━━━━━━ [总时长]
           ↓
[🔁] [⏮️] [▶️ 大按钮] [⏭️] [🔊]
```

## 交互功能

### 1. 播放控制
- **循环模式**：点击左侧图标切换
  - 🔁 循环全部
  - 🔂 单曲循环
  - 🔀 随机播放

### 2. 音量控制
- 悬停在音量图标上显示滑块
- 垂直滑块设计
- 实时音量调节

### 3. 设置面板
- 点击"更多"按钮打开
- 倍速控制滑块
- 音调保持开关

### 4. 进度条
- 点击跳转
- 拖动调节
- 实时显示当前时间

## 主题适配

专辑模式完全继承主界面的主题色：
- 主按钮使用歌曲主题色
- 进度条使用主题色填充
- 音量滑块使用主题色
- 按钮悬停效果统一

## 动画效果

1. **封面动画**
   - 播放时轻微放大（scale: 1.0）
   - 暂停时轻微缩小（scale: 0.98）
   - 悬停时上浮效果

2. **按钮动画**
   - 悬停放大 5%
   - 点击缩小 5%
   - 平滑过渡

3. **音量滑块**
   - 滑入动画
   - 0.2s 缓动

## 响应式断点

- **桌面端** (>768px)：完整尺寸
- **平板端** (480-768px)：中等尺寸
- **手机端** (<480px)：紧凑布局
