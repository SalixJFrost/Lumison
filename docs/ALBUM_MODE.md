# 专辑模式 (Album Mode)

## 概述

专辑模式是一个简洁、现代的音乐播放界面，专注于专辑封面展示，UI风格与主界面保持一致。

## 特性

- **超大尺寸专辑封面**：480x480px（桌面端），响应式设计
- **完整的播放控制**：
  - 播放/暂停（大型主按钮）
  - 上一首/下一首
  - 播放模式切换（循环、单曲循环、随机）
  - 音量控制（悬停显示滑块）
- **歌曲信息展示**：标题、艺术家
- **操作按钮**：喜欢、更多设置
- **进度条和时间显示**
- **设置面板**：倍速控制、音调保持
- **毛玻璃效果**：与主界面一致的视觉风格
- **流畅动画**：基于 react-spring
- **响应式设计**：完美支持桌面和移动端

## 使用方法

1. 点击顶部栏的"设置"按钮（齿轮图标）
2. 在设置面板中找到"视图模式"选项
3. 选择"专辑"模式
4. 界面将切换到专辑模式视图

## 功能说明

### 播放控制
- **播放模式按钮**（左侧）：循环播放 → 单曲循环 → 随机播放
- **上一首/下一首**：快速切换歌曲
- **播放/暂停**：中央大按钮，带主题色高亮
- **音量控制**（右侧）：悬停显示音量滑块

### 设置面板
- **倍速控制**：0.5x - 2.0x，步进 0.25x
- **音调保持**：保持原始音调

## 切换回默认模式

在设置面板的"视图模式"中选择"默认"即可切换回原来的双栏布局。

## 技术实现

- 使用 React + TypeScript
- 基于 react-spring 实现流畅动画
- CSS 模块化设计，毛玻璃效果
- 完整的国际化支持（中英文）
- 响应式布局，适配各种屏幕尺寸
