# Lumison v1.0.2 Release Notes

## 🎉 新功能

### 🎨 全新应用图标
- 采用耳机 + 光带设计，简洁现代
- 银白金属质感配合暖白色光带
- 光带末端形成 "L" 字母，强化品牌标识
- 已应用到所有平台（Windows、macOS、Linux、Android、iOS）

### 🎵 歌词系统优化
- **网易云歌词优先**：自动优先从网易云音乐 API 获取歌词
- 支持逐字歌词和翻译
- 删除了 QQ 音乐和酷狗音乐 API（因 CORS 问题）
- 保留第三方歌词 API 作为备用

### 📊 可视化器修复
- 修复了可视化器加载问题
- 改进了 analyser 初始化逻辑
- 添加了状态追踪，确保可视化器正常显示
- 默认启用可视化器

### 🎨 UI 改进
- TopBar Logo 保持简洁风格
- 移除了不必要的图标和特效
- 统一的视觉风格

## 🔧 技术改进

### 图标生成系统
- 新增自动化图标生成脚本
- 支持从 SVG 一键生成所有平台图标
- 新增命令：
  - `npm run generate:icon` - 生成 PNG 图标
  - `npm run generate:all-icons` - 一键生成所有平台图标

### 代码优化
- 清理了未使用的文件和代码
- 改进了性能和内存使用
- 更好的错误处理和日志记录

## 📦 安装包

### Windows
- **NSIS 安装程序**（推荐）：`Lumison_1.0.2_x64-setup.exe`
- **MSI 安装程序**：`Lumison_1.0.2_x64_en-US.msi`

### 系统要求
- Windows 10/11 (64位)
- 最低 4GB RAM
- 100MB 可用磁盘空间

## 🐛 Bug 修复

- 修复了可视化器不显示的问题
- 修复了 spatial engine 初始化时序问题
- 改进了歌词获取的稳定性
- 修复了多个 UI 渲染问题

## 📝 已知问题

- 安装包未签名，Windows 可能显示"未知发布者"警告（可安全忽略）
- 部分歌曲可能无法从网易云获取歌词（会自动尝试第三方 API）

## 🙏 致谢

感谢所有用户的反馈和建议！
