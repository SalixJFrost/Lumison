# 主题切换动画优化

## 更新内容

### 1. 过渡时长调整
- 从 0.6秒 延长到 1.2秒
- 让颜色变化更加从容自然

### 2. 缓动函数优化
- 从 `cubic-bezier(0.4, 0, 0.2, 1)` 改为 `cubic-bezier(0.25, 0.1, 0.25, 1)`
- 使用更平滑的 ease-in-out 曲线
- 减少突兀感，增加流畅度

### 3. 背景渐变插值
- 添加颜色插值系统
- 背景颜色在主题切换时平滑过渡
- 使用 ease-out cubic 缓动函数
- 与 CSS 过渡时间同步（1.2秒）

### 4. 控件过渡优化
- 播放/暂停按钮：背景色和文字色平滑过渡
- 进度条：颜色变化更加自然
- 前进/后退按钮：图标颜色柔和过渡
- 所有交互动画保持快速响应（0.2秒）

## 技术实现

### CSS 过渡
```css
transition: color 1.2s cubic-bezier(0.25, 0.1, 0.25, 1);
```

### Canvas 颜色插值
- 实时计算源颜色和目标颜色之间的中间值
- 使用 ease-out cubic 函数平滑过渡
- 每帧更新渐变，确保流畅

## 效果
- 主题切换更加优雅
- 颜色变化不再仓促
- 视觉体验更加舒适
- 保持了交互的即时响应
